## Project Overview
The Mobile Sales Dashboard is an interactive data visualization project created using Power BI.  
It provides a detailed analysis of mobile sales across various brands, models, and regions.  
The goal is to help businesses track performance, revenue, profit, and identify best-selling models effectively.

## Brief Information
This project transforms raw mobile sales data into clear business insights.  
It focuses on:
- Total revenue and profit performance  
- Brand-wise and region-wise sales analysis  
- Month-wise and yearly growth patterns  
- Identifying top-performing models  

## Tools and Technologies
- Power BI – For dashboard visualization  
- Microsoft Excel – For data cleaning and preparation  

## Key Insights
- Sales trends over time  
- Brand and model comparisons  
- Regional performance analysis  
- Profit and revenue visualization  

## Objective
To create an easy-to-use Power BI dashboard that helps businesses monitor mobile sales and improve decision-making through visual insights.
